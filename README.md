# Shopping Basket App

Shopping basket demonstrating MVVM architecture with Hilt DI.

## Features
- Fetch products from API
- Add products to basket
- Calculate total (cost_price preferred, retail_price fallback)
- Clear basket

## Tech Stack
- MVVM + Hilt
- Retrofit + Gson
- RecyclerView

## Setup
1. Clone repo
2. Open in Android Studio
3. Run app
